# Airline Operations: Root-Cause & Propagation Analysis
## Project Overview
This project analyzes 5M+ flight records to identify why delays happen and how a single late flight ripples through an entire day of operations.

## Tech Stack
- SQL (MySQL): Aggregation and high-level KPI extraction.

## Repository Structure
data_source.txt: https://mavenanalytics.io/data-playground/airline-flight-delays
